# Pong Game

This is a simple Pong game implemented in HTML, CSS, and JavaScript.

## Features
- Player controls the left paddle using arrow keys (Up/Down)
- Or, control the left paddle using the mouse (y-position)
- Computer AI controls the right paddle
- Ball with realistic bouncing and collision detection
- Scoreboard tracks player and computer points
- Game starts immediately and restarts automatically
- Easy to customize paddles, speed, and more

## How to Play
1. **Download** or **clone** this repository.
2. Open the `index.html` file in your web browser.
3. Use the **⬆️ Up Arrow** and **⬇️ Down Arrow** keys to move your paddle up and down.
    - Or use your mouse to control the left paddle.
4. The computer controls the right paddle, try to beat it!
5. The first to reach your score goal (or keep playing for fun) wins. The ball resets after a score.
